== Description ==

Get Education is the child theme of Bizberg WordPress Theme. The theme is a clean & modern Education WordPress Theme. Get Education is suitable for any tuition, marketing, bookselling, university, workshop, college, school, course hub, ecommerce, woocommerce, store, shop, training center, blogging, or educational institution. Also, the Get Education theme is a perfect solution for your educational websites. This Free WordPress theme is fully responsive, cross-browser compatible, translation ready & SEO friendly. If you run into any problems while using our theme, you can refer to extensive documentation or contact our friendly support team. It fully supports Elementor, and all the sections are made from Elementor.
